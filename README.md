# 🌍 Bangladesh Climate & Education Data Lab

An interactive data analysis project exploring the relationship between **climate risks** and **education inequality** across Bangladesh.


## 📊 Features

* Climate Risk vs School Access Analysis
* Flood Risk Visualization
* Cyclone Exposure Mapping
* Education Inequality Dashboard
* Population Risk Analysis


## 🛠 Tools Used

* Python
* Streamlit
* Pandas
* Plotly
* Folium


## 📁 Project Structure

data/ → CSV datasets containing climate and education data
code/ → Python scripts for analysis and visualization
Maps/ → Interactive HTML maps
app.py → Main dashboard application


## 📌 Key Insights

* Coastal regions show higher exposure to cyclones
* Flood-prone districts tend to have lower school access
* Climate vulnerability is linked with education inequality


## 🎯 Purpose

This project demonstrates how data visualization and analytics can be used to understand the impact of climate risks on education systems in Bangladesh.
